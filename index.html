Exploring the <span> Tag

Inline Styles and Semantics

The <span> tag is an inline container used to mark up a part of a text, or a part of a document. It's used to apply styling via CSS or to manipulate with JavaScript.

Here are some examples of what the <span> tag can do:

  • Change color: This text is red.

  • Add background: This text has a lightblue background.

  • Italicize text: This text is italic.

  • Combine effects: This text combines all three effects.
